🔥 Firebase Setup Instructions

⚠️ Action Required — Takes ~10 minutes Follow these steps once to connect Firebase. After that, all dynamic features (nurse availability, track records, reviews) work automatically.

Step 1: Go to console.firebase.google.com → Sign in with your Google account.

Step 2: Click "Add Project" → Name it "tirupati-ambulance" → Disable Google Analytics (optional) → Create project.

Step 3: Click the Web icon (</>) → Register app as "tirupati-web" → Copy the firebaseConfig object shown.

Step 4: In the left menu → Build → Realtime Database → Create Database → Choose Start in test mode → Enable.

Step 5: In the HTML file, find the comment // PASTE YOUR FIREBASE CONFIG HERE and replace the placeholder config with the one you copied in Step 3.

Step 6: In Firebase Console → Realtime Database → Rules → paste this rule and Publish:

{
  "rules": {
    "nurses":       { ".read": true,  ".write": "auth != null || true" },
    "trackRecords": { ".read": true,  ".write": "auth != null || true" },
    "reviews":      { ".read": true,  ".write": true }
  }
}

Step 7: Save and reload your website. The admin panel will now read/write live data!

Note: The admin password is currently admin123. To change it, search for ADMIN_PASSWORD in the HTML file and update the value.
